# COCOSiL 基軸資料 INDEX v2

## L0
**COCOSiLプロジェクトの全基軸資料を統合し、最新版へのルーティングとチーム体制・アクションアイテムを管理するSingle Source of Truth。**

---

## L1
- **設計原則**: Drift-Aware（実装との乖離確認）、Single Source of Truth、Reader-First。
- **チーム体制**: えんまさ（意味設計）、ヒラメ（構造設計/API）、まあみ（見た目設計/UI）の3名体制。
- **資料一覧**: 戦略・事業系、開発系、リサーチ系、オペレーション系の全最新ファイルを網羅。
- **読み分けガイド**: シチュエーション別に「誰が・いつ・どの資料を読むべきか」を定義。
- **アクションアイテム**: Gamma API制限確認、DBスキーマ確定など、現在の未解決課題と優先度を管理。

| 原則 | 内容 |
|------|------|

| メンバー | 役割 | 担当レイヤー |

| **えんまさ** | CEO・言語設計オーナー | 意味設計（言語・AIプロンプト・ナレッジベース・品質判断） |
