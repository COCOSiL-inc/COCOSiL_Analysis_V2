---

## 2. 解決する課題

### 本質課題 ——「現代の無明」

現代人は対人関係で繰り返し消耗しています。
その根本原因は「**自分も相手も、なぜこうなるかが分からない**」という自己・他者への無知にあります。

これは職場・恋愛・家族・友人、あらゆる対人関係で共通して発現します。

### 既存サービスの限界

| カテゴリ | できること | できないこと |
|---------|-----------|-------------|
| 共感系AI（チャッピー等） | 話を聞いてもらえる | 根拠がなく、何も変わらない |
| 診断系（MBTI等） | 自己分析の枠組みを得られる | 感情に届かず、行動に繋がらない |
| 対面占い師 | 深い共感と洞察を得られる | 月1回・高コスト・29日間の空白がある |

### 市場の空白

「**感情処理 → 自己理解 → 行動変容**」を一気通貫で届けるサービスが、現時点で存在しません。

---

## 3. COCOSiLのソリューション

### 核心体験 ——「腑落ち体験」

COCOSiLが届けるのは単なる共感でも表面的な自己分析でもなく、
**「なぜ自分はこうなのか」が腑落ちし、本当にラクになれる体験**です。

### 設計の鉄則（非交渉のUXシーケンス）

**共感 → 安心 → 分析 → 行動**

この順序を崩すとユーザーは離脱する——複数のユーザーインタビューで検証済みの設計原則。

### 3つのコア機能

**① 共感AIチャット（MVPの心臓部）**

傾聴→深掘り→行動提案の3フェーズ設計。
性格分析データに基づく「根拠のある共感」で、一般的なAIチャットとの決定的な差別化を実現。

**② 4体系統合レポート**

MBTI・西洋占星術・動物占い（60アニマル）・六星占術の4フレームワークをAIで統合。
複数視点を横断することで、単一手法では見えない自己像を可視化。

**③ パーソナライズドアクションプラン**

統合診断結果に基づき、その人固有の行動提案を生成。「わかった」で終わらせず「変わった」につなげる設計。

### ポジショニング

占い師の「共感」、心理学の「分析」、AIの「パーソナライズ」——この3つを初めて統合したサービス。
