# COCOSiL 言語設計文書 v1.0

## L0
**「占い」という言葉による抵抗感を排除し、COCOSiLを自己理解・性格分析サービスとして認知させるための、全レイヤー共通の言語設計ルールブック。**

---

## L1
- **目的**: ターゲット層の「占い」への冷笑感・抵抗感を回避し、普遍的な自己理解ツールとして位置づける。
- **使用禁止ワード**: 「占い」「占い師」「鑑定」「運勢」「当たる」等を一律禁止とし、代替表現（性格分析、AIパートナー等）を定義。
- **トーン＆マナー**: スピリチュアル感や権威性を排除し、内省的で根拠のある共感、小さな肯定を積み重ねるトーンを目指す。
- **UI適用例**: オンボーディング、レポート、AIチャット、アクション記録など、各画面でのNG/推奨コピーを規定。
- **AIプロンプト**: プロンプト内でも禁止ワードを適用し、システム全体での一貫性を担保する。

## なぜこの文書が必要か

COCOSiLの機能には「六星占術」「動物占い」「星座分析」など、**占術・占いに由来するコンテンツ**が含まれる。しかし、25〜35歳のターゲット層の中には「占い」という言葉に抵抗感・冷笑感を持つ層が存在する。

一方、同じ層でも「MBTI」「星座」「性格診断」は日常会話で普通に使われる。

COCOSiLは「占いアプリ」ではなく「**自己理解・性格分析AIサービス**」として認知されることを目指す。この目標のために、言葉の選択を全チームで統一する。

## 1. 使用禁止ワード → 代替表現テーブル

| ❌ 使用禁止 | ✅ 代替表現（文脈によって選択） | 使用場面の例 |
|-----------|--------------------------|------------|
| **占い** | 性格分析 / 自己理解 / パーソナリティ診断 / 自分を知る | 機能説明・LP |
| **占い師** | AIパートナー / 自己理解アシスタント | 役割説明 |
| **鑑定** | 診断 / 分析 / あなたを読み解く | 動詞として使う場合 |
| **鑑定結果** | 診断結果 / あなたの分析レポート | 結果画面 |
| **運勢** | 傾向 / パターン / 特性 / あなたの性質 | 特徴描写 |
| **占星術** | 星座分析 / 星座ベースの性格傾向 | 機能説明 |
| **当たる / 当たった** | 腑に落ちる / ぴったりくる / 自分らしい | 体験表現 |
| **信じる / 信じている** | 参考にする / 使っている | ユーザー発話文脈 |
| **霊感** | （使用禁止・代替なし） | — |
| **霊視** | （使用禁止・代替なし） | — |

### 条件付き使用（文脈によって判断）

| 言葉 | 使用可能な文脈 | 注意点 |
|------|-------------|--------|
| **六星占術** | コンテンツ名として使う場合は原語のまま使用可 | 「診断方法のひとつ」として紹介する枠の中に収める |
| **動物占い** | 同上 | 「60アニマル診断」を優先表記 |
| **星座** | 日常語として定着しているため使用可 | 「占い」と抱き合わせるのは避ける |
| **MBTI** | 固有名詞として使用可 | 「性格診断ツール」という説明を初出時に添える |

## 2. COCOSiLの言葉の世界観

### プロダクトを表すフレーズ

| 場面 | フレーズ |
|------|---------|
| ひとことで | 自分を知って、ラクになる |
| LP主軸 | 自分が分かると、人間関係がちょっとラクになる。 |
| 若年層向け | 夜ひとりで抱えてること、ここに置いてきて。 |
| 競合差別化 | 話して、わかって、変わる。 |
| ポジショニング | 自分の取扱説明書を一緒に読んでくれるAI |

### 避けるべきトーン

- 「スピリチュアル」感が強い表現（「宇宙の意志」「前世」「カルマ」等）
- 「未来を予言する」ニュアンス（「あなたの運命は〜」「今月の運気は〜」）
- 「信じさせる」型の権威表現（「プロの占い師が〜」「2000年の歴史が〜」）
- 過度なポジティブ強制（「絶対うまくいきます！」「大吉です！」）

### 目指すトーン

- **内省的・落ち着いた温かさ**（「あなたのこと、少し分かった気がします」）
- **根拠のある共感**（「INFJタイプのあなたは、こういう場面で特に深く感じるんですよね」）
- **小さな肯定の積み重ね**（「それ、すごく自然な気持ちですよ」）
- **行動できる具体性**（「明日、まず〇〇してみませんか？」）

## 3. 画面別コピーライティング例

### オンボーディング

| 画面 | ❌ NG表現 | ✅ 推奨表現 |
|------|---------|----------|
| ようこそ画面 | 「あなたの運命を占います」 | 「あなたのことを、もっとよく知らせてください」 |
| 生年月日入力 | 「誕生日で運命が変わります」 | 「あなたの性格タイプを分析するために使います」 |
| MBTI診断開始 | 「占い診断スタート！」 | 「性格診断をはじめましょう」 |
| 診断完了 | 「鑑定が完了しました」 | 「あなたのタイプが分かりました」 |

### 統合レポート

| 場面 | ❌ NG表現 | ✅ 推奨表現 |
|------|---------|----------|
| ローディング中 | 「占い結果を生成中...」 | 「あなたの分析レポートを作っています」 |
| レポートタイトル | 「あなたの運命鑑定書」 | 「あなたの性格分析レポート」 |
| 満足度質問 | 「占いは当たりましたか？」 | 「この分析は、あなたに当てはまりましたか？」 |

### AIチャット

| フェーズ | ❌ NG表現 | ✅ 推奨表現 |
|---------|---------|----------|
| 開始 | 「今日は何を占いましょうか？」 | 「今日、何か話したいことはありますか？」 |
| 分析フェーズ | 「あなたの星が示すのは...」 | 「あなたのINFJタイプだと、こういう場面で特につらく感じやすいんです」 |
| アクション提案 | 「運気が上がる行動は〜」 | 「あなたの性格タイプに合った方法で、こんなことを試してみませんか？」 |

### アクション記録（プレッシャーゼロ設計）

| 場面 | ❌ NG表現 | ✅ 推奨表現 |
|------|---------|----------|
| 空状態 | 「まだアクションが完了していません」 | 「できたこと、ここに記録していこう」 |
| 達成ボタン | 「チェック / 完了」 | 「できた！」 |
| 達成後 | 「残り〇個あります」 | 「よくできました。また来たね。」 |
| 未達成表示 | 「未完了: 3件」 | （表示しない） |

## 4. AIプロンプトへの適用

### 禁止ワード（プロンプト内でも使用禁止）

```
禁止: 占い、鑑定、運勢、当たる、占星術（単体での使用）、霊感
禁止: 「あなたの星が示すのは」「運命では」「前世から」
```

### 推奨フレーズ（プロンプト内での言い回し）

```
推奨: 「〇〇タイプのあなたは、こういう傾向があります」
推奨: 「性格分析から見ると、〇〇という特性があなたにはあります」
推奨: 「これはあなたのタイプに特有のパターンです」
推奨: 「腑に落ちますか？」「自分らしいと感じますか？」
```
