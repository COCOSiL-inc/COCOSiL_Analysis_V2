# COCOSiL リーンキャンバス v3.0

## L0
**感情処理・自己理解・行動変容を一気通貫で提供し、月額12万円（150名）の損益分岐を目指すCOCOSiLの最新ビジネスモデル。**

---

## L1
- **課題とUVP**: 現代の「無明（対人関係の消耗）」に対し、東洋哲学・西洋心理学・AIを統合した「腑落ち体験」を提供する。
- **ターゲット**: 25〜35歳の若手社会人で、対人関係に繰り返し消耗している層。
- **ソリューション**: 4体系（MBTI・星座・動物・六星占術）の統合診断と、根拠ある共感AIチャット。
- **コスト・KPI**: 損益分岐は月額12万円。コアKPIは7日以内再訪率30%以上。
- **優位性**: 占い師の共感、心理学の分析、AIのパーソナライズを組み合わせた独自のUXシーケンス（共感→安心→分析→行動）。

## ① 課題

### 本質課題 ／ 現代の「無明」

- 自分も相手も「なぜこうなるか」が分からず、同じ対人パターンで繰り返し消耗する
- 「自己への無知」こそが苦の根源——東洋哲学2500年の診断と一致する普遍的ペイン

### 課題の発現シーン（文脈）

職場の対人関係 ／ 恋愛・パートナー ／ 家族関係 ／ 友人・コミュニティ

### 既存代替手段の限界

| 代替手段 | 提供するもの | 欠けているもの |
|---------|-----------|--------------|
| 共感系AI（チャッピー等） | 話せる・感情の受け皿 | 根拠がなく何も変わらない |
| 診断系（MBTI等） | 自己分析の枠組み | 感情に届かず行動に繋がらない |
| 対面占い師 | 深い共感＋洞察 | 月¥1万・非日常・29日間の空白 |

### 市場の空白

「感情処理 → 自己理解 → 行動変容」を一気通貫で提供するサービスが存在しない

## ピッチナラティブ骨格

**Opening：**
「人類は数千年間、『なぜ自分は苦しいのか』を問い続けてきた。仏教は『無明（自己への無知）』が苦の根源だと答えた。東洋哲学の到達した知恵をAIで日常に届けるのが、COCOSiLです。」

**Problem：**
「現代人は対人関係で消耗し続けている。吐き出す場所はある。分析するツールもある。でも『なぜ自分はこうなのか』が腑落ちして、本当にラクになれるサービスはない。」

**Solution：**
「COCOSiLは感情処理→自己理解→行動変容を一気通貫で提供する。占い師の共感、心理学の分析、AIのパーソナライズを統合した『自分の取扱説明書』。」

**Why Now：**
「LLMの登場で、初めて『感情に寄り添いながら構造的な自己理解を促す』体験が設計可能になった。東洋哲学の到達点をAIが民主化できる、最初の機会がここにある。」
