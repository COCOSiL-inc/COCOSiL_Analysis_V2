# COCOSiL 言語設計ルール

## 核心原則

COCOSiLは「共感AI × 多角的性格分析」プロダクトであり、占いサービスではない。
ユーザーに「心理学・統計的裏付けのある、根拠のある共感」と感じさせるトーンを維持する。

## 禁止ワード → 代替表現テーブル

| 禁止語 | 代替表現 | 禁止の理由 |
|--------|---------|-----------|
| 占い | 性格分析・パーソナリティ診断・統合レポート | スピリチュアルな印象、誤解を招く |
| 鑑定 | 分析・レポート・診断 | 占い師による権威的な判断を連想させる |
| 運勢 | 傾向・特徴・パターン・強み | 未来予測の印象を避ける |
| 占い師 | 分析パートナー・AIアドバイザー | 人間の占い師との混同を避ける |
| 当たる / 当たった | 共感できる・理解された・しっくりくる | 正否・的中という概念を避ける |
| 霊感 / 霊視 | （代替なし — 使用禁止） | スピリチュアルコンテンツとの混同を完全に防ぐ |
| 占術 | 〇〇分析・〇〇タイプ判定 | 術（アート）ではなく分析の印象に統一 |

## 例外: 内部テクニカルコンテキスト

コード内の**変数名・DB カラム名・型定義・内部ドキュメント**では、
「動物占い」「六星占術」等の正式名称をそのまま使用してよい。

禁止対象は**ユーザーが目にする文言**のみ:
- UI テキスト
- AI の出力（システムプロンプト・レスポンス）
- SNS シェアカードのコピー
- エラーメッセージ・ツールチップ
- LP・告知文

## 目指すトーン

- **温かみがあるが、媚びない** — 友人でも先生でもなく「信頼できる相談相手」
- **根拠のある共感** — データと心理学に基づいているという安心感を伝える
- **断定より傾向** — 「あなたは〜です」より「〜な傾向があります」「〜を大切にしていそうです」
- **肯定ファースト** — 弱点を伝える前に、強みや価値を先に示す

## AI プロンプトへの適用

システムプロンプト・ユーザー向けメッセージを書く際:
1. 上記テーブルの禁止語が含まれていないか確認する
2. 「占いの結果として〜」ではなく「あなたのタイプから読み取れるのは〜」と表現する
3. チャットの3フェーズ（共感 → 深掘り → アクション）でも同じトーンを維持する

## ブランドフレーズ（確定版）

UIコピー・LP・SNS投稿で使える確定フレーズ。これ以外の言い回しは使用前にえんまさに確認する。

| 用途 | フレーズ |
|---|---|
| ワンライナー | 「自分を知って、ラクになる」 |
| LP主軸コピー | 「自分が分かると、人間関係がちょっとラクになる。」 |
| 差別化フレーズ | 「話して、わかって、変わる。」 |
| 深夜感情発散層向け | 「夜ひとりで抱えてること、ここに置いてきて。」 |
| UVP（投資家・B2B向け） | 「占い師の共感 × 心理学の分析 × AIのパーソナライズを初めて統合」 |

## ターゲット別トーン指針

| ターゲット層 | 特徴 | トーンの方向 |
|---|---|---|
| 25-35歳社会人（分析系） | MBTIに詳しい・SNSで自己分析をシェアする | 根拠ある共感・落ち着いた温かさ・データ感 |
| 18-24歳深夜発散層 | 夜ひとりで感情処理・承認欲求が強い | 親しみやすく・夜の雰囲気・「あなたのことを分かってる」感 |
| 35-45歳ヘビーユーザー | 既存の性格診断・統合レポートに馴染みあり | 深さ・網羅感・「ここまで分かるのか」という驚き |

## 正しい表現例

| NG | OK |
|----|----|
| 「今日の運勢は吉です」 | 「あなたのタイプは今、〇〇な傾向があります」 |
| 「占い結果: INFJ × 蟹座 × 子守熊」 | 「統合レポート: INFJ × 蟹座 × 子守熊タイプ」 |
| 「占いが当たった！」 | 「すごく共感できる」「しっくりきた」 |
| 「あなたの鑑定をします」 | 「あなたのパーソナリティを分析します」 |
