# COCOSiL ドメインスキル

## プロダクト概要

- **サービス名：** COCOSiL（ここしる）
- **コンセプト：** 「自分を知って、ラクになる」
- **機能：** 共感AIチャット × MBTI・星座・動物性格診断（60アニマル）・六星占術の4体系統合性格分析
- **ターゲット：** 25〜35歳の若手社会人。対人関係に繰り返し消耗している層
- **PMF仮説：** 7日以内再訪率30%以上

## ビジネスモデル & PMF

- **PMF仮説：** 7日以内再訪率 **30%以上**（継続利用意欲の主要KPI）
- **損益分岐：** 有料ユーザー **150人・月額¥12万**
- **料金プラン：** 無料（月5回チャット） / ライト¥500 / スタンダード¥1,500
- **3年後目標：** MAU 10,000人・月間売上¥200万

### フェーズロードマップ

| フェーズ | 期間 | 目標 |
|---|---|---|
| Phase 1 | 0-6ヶ月 | 新規500人/月、無料→有料転換率10% |
| Phase 2 | 6-12ヶ月 | 継続率60%、転換率15% |
| Phase 3 | 12ヶ月〜 | LTV/CAC≥3.0、MAU 5,000人 |

## 依存サービスリスク

| サービス | 機能 | リスク |
|---|---|---|
| **Gamma API** | F2 統合レポート生成 | レート制限・コスト未確認。Sprint 3前に必ず調査すること |
| **OpenAI** | F3 共感AIチャット | チャットコスト。ユーザー1セッションあたりの上限設計が必要 |
| **Clerk** | F4 認証・オンボーディング | 無料枠上限（MAU 10,000）。大規模化時に課金発生 |

## 絶対に守るUXシーケンス

```
共感 → 安心 → 分析 → 行動
```

この順序は絶対に変えない（非交渉）。

| フェーズ | ユーザー状態 | AIの役割 | やってはいけないこと |
|---|---|---|---|
| 共感 | 悩みを打ち明ける段階 | 感情を受け取り、承認する | 解決策・診断・アドバイスを先出し |
| 安心 | 「分かってもらえた」と感じる段階 | 「あなたの気持ちは自然です」を伝える | 問題の深刻化・分析への急転換 |
| 分析 | 自分を客観的に見る準備ができた段階 | 性格タイプを使って構造的に説明する | 性格に合わない行動を提案 |
| 行動 | 「やってみよう」と思える段階 | 小さく実行できるアクションを1〜3個提示 | 内向型に「飲み会で積極的に話しかけて」等のNG提案 |

## 4体系の役割分担

| 体系 | 何を測るか | レポートでの使い方 |
|---|---|---|
| MBTI（4文字） | 認知・判断の基本パターン | 思考スタイル・コミュニケーション傾向 |
| 星座（12） | 感情・関係性の傾向 | 対人関係・感情処理スタイル |
| 動物性格診断（60アニマル） | 行動パターン・価値観の細分化 | MBTIとの矛盾の架け橋・行動傾向 |
| 六星占術（12） | 運命サイクル・本来の気質 | 人生観・エネルギーの使い方 |

## アンチパターン集（絶対やらない）

- 最初のメッセージで性格分析結果を見せる
- 「あなたはINFJなので〜してください」という断定的アドバイス
- 性格タイプと相性が悪い行動を提案する（例：内向型に外向的行動を強要）
- 統合レポートを診断フォーム直後に即表示する（共感フェーズをスキップ → 7日再訪率低下）
- ユーザーがまだ悩みを話していない段階でアクションプランを出す
- Gamma APIのレート制限を事前確認せずにSprint 3の統合レポート実装を始める
- 「動物占い」「六星占術」という正式名称をUIテキストに使う（コード内のみ許容）

## ドメイン言語ルール（v1.3）

UIコピー・AIプロンプト・エラーメッセージを書く際は `language-design` スキルを参照すること。

> 🔒 **正は `lib/constitution/banned-words.ts`**（Constitution as Code）。本テーブルと文書 `language-design-v1.md` は解説。ドリフト時はコードを正とする。

| 使用禁止 | 代替表現 |
|---------|---------|
| 占い | 性格分析 / 自己理解 / パーソナリティ診断 |
| 占い師 | カウンセラー / セラピスト / AIパートナー |
| 鑑定 | 診断 / 分析 / 読み解き |
| 運勢 | 傾向 / パターン / 特性 |
| 占星術 | 星座分析 / 星座ベースの性格傾向 |
| 動物占い | 動物性格診断 / 60アニマル診断 |
| 当たる | 自分らしい / 腑に落ちる / ぴったりくる |
| 霊感・霊視 | （使用不可・代替なし） |

**例外：** コード内の変数名・DBカラム名では正式名称（動物占い、六星占術）の使用可。
